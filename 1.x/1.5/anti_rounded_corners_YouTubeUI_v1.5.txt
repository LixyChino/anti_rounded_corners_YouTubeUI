/* v1.0:2024年2月29日 YouTubeが2022年10月ごろから始めてしまった角丸UIを粛清すべく、Firefox BetaとStyleBot(スマホ版Firefox BetaはStyleBotに対応していないので動作はStylus)とuBlock Origin(広告ブロッカーは無効の状態で)の要素抹消機能を使ってUIをマシにしていこうということで始まりました。大体border-radiusを無効にしていってます。 */
/* v.1.1:2024年3月7日 Webアプリ版特有のコメントの激狭返信欄を広くしました。あとはチャンネル登録ボタンと高評価ボタンたちの位置を17.10くらいまでの神UI時代から見て正しい位置に修正しました。 */
/* v1.2:2024年3月8日 概要欄のもっと見るを消しました。つまり概要欄を開けばすぐに全部見えます。あとは全画面表示時の「border-radius」を粛清しました。 */
/* v1.3:2024年3月9日 概要欄のもっと見るを消しきれない問題を修復しました。あとはチャンネル登録者数の表示部に「登録者数」と「人」を追加しました。 */
/* v1.4:2024年3月9日 再生リストのダサい背景を無効にしました。あとは「「広告を」スキップ」にして、文字もデカくしました。概要欄のもっと見るを消しきれない問題、今度こそ解決できるのか楽しみ。←解決できませんでした */
/* v1.5:2024年3月12日 ついに概要欄のもっと見るを消すことに成功しました。広告をスキップがバグる問題も解決。スポンサーとかかっこつけた表記にしないでよ。元の広告表記に戻してあげました。再生位置を移動するときのサムネ表示たちの角丸も粛清しました。チャンネル登録ボタンもシンプルになりました。遂に無理やり感が凄かった「登録者数」がちゃんとした「チャンネル登録者数」になりました。Twitter(旧𝕏)の𝕏表記をTwitterに正しました。ボタン配置をYouTubeアプリバージョン17.10までの神ボタン配置を無理やり感凄いけど再現してみました。全画面の倒し残し角丸を倒しました。本バージョンから開発にStyleBotではなくStylusを使用することにしました。どうやら微妙に挙動が違うようです。*/
